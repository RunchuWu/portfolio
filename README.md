# Runchu's Portfolio

**Version 1**  
**Date:** March 24, 2026

This repository contains the first version of my personal portfolio website. It introduces me as Runchu Wu, an AI Product Manager and Full-stack Developer at Duke University, and highlights the work, experiences, and interests that shape how I build products.

## Overview

The site is a bilingual personal portfolio built in English and Chinese. It is designed to present both professional work and personal identity in one place, with sections for featured projects, life outside of work, and a resume-style summary of experience, education, and skills.

## What’s Included

- A home page with a personal introduction and portfolio navigation
- A work section featuring projects such as Liberata, Solvi.ai, and AI Tarot Reader
- A life section covering music, tennis, education, and social work
- A resume section with experience, projects, education, and skills
- A language toggle for English and Chinese

## Tech Stack

- React 19
- TypeScript
- Vite
- Motion
- Lucide React
- Custom CSS

## Run Locally

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Build for production:
   `npm run build`
4. Preview the production build:
   `npm run preview`
